# Scraper Project

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/BeardedCyberDad-ops/Scraper-1.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Scraper-1
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage Guide
To run the scraper, use the following command:
```bash
node scraper.js
```

## Learning Path for Milestone 1
1. **Understanding Web Scraping Basics**  
   Learn what web scraping is and the different tools available.
2. **Introduction to Node.js**  
   Familiarize yourself with Node.js as the runtime environment you'll use.
3. **Using Axios for HTTP Requests**  
   Understand how to fetch data from websites using Axios.
4. **Parsing HTML with Cheerio**  
   Learn how to parse and manipulate HTML data using Cheerio.
5. **Storing Data**  
   Explore different methods of storing the scraped data (e.g., JSON, databases).
